## 1. 项目

Manim 动画项目，用于帮助人提升认知的视频，每个 Python 文件是独立场景
环境：必须用虚拟环境，Python 3.14.2 + Manim 0.20.1，执行 source venv/bin/activate
渲染：-ql 快速预览（480p15），-qh 生产质量（1080p60）

## 2.视频规范

### 结构要求

#### 1.片头

- 大标题，字号60，颜色#FFC000，视频到达正片时，上居中字号变为30
- 动画：文字出场，出场结束后圆环粒子炸开的动画 
- 位置：居中

#### 2.正片

- 左右结构，左70%文字，右30%动画
- 标题：位置左上，字号46，颜色 #FFD700（金色），**必须加粗** `weight=BOLD`
- 正文: 字号32 ，颜色纯白
- 动画：大致符合左边标题内容即可，动画需要有动感，需要一直在动
  - **入场顺序**：标题 → 描述 → 右边动画（子元素依次入场）
  - **动画风格**：丝滑流畅，避免过度复杂的堆砌
  - **动画时长**：每个特征约 1.8-2 秒
  - **运动方式**：使用 `there_and_back` 让动画有节奏感
  - **元素数量**：每个特征 3-5 个核心元素即可

#### 3.片尾

### 视频制作流程

确定选题 - 规划视频 - 确定规划 - 开始编码 - git提交代码

### 其他要求

写manim代码请加载 skill manim 

视频时长1分钟

代码写在一个文件中，代码分阶段写入。

背景 纯黑背景白色细线网格

字体宋体（Songti SC），**标题和正文必须加粗** `weight=BOLD`

## 3.配色方案

### 方案1：黑金奢华（推荐）

| 元素 | 颜色 | 代码 |
|-----|------|-----|
| 主标题 | 金色 | `#FFD700` |
| 正片标题 | 金色 **加粗** | `#FFD700` + `weight=BOLD` |
| 描述文字 | 纯白 **加粗** | `#FFFFFF` + `weight=BOLD` |
| 放射线 | 金色系渐变 | `#FFD700`, `#FFA500`, `#FF8C00`, `#D4AF37` |
| 背景 | 纯黑 | `BLACK` |
| 网格 | 深灰细线 | `#333333` |

### 其他配色方案
